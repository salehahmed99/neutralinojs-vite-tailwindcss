# neutralinojs-vite-tailwindcss

A modern React template for building Neutralinojs apps using TypeScript, Vite, and Tailwind CSS.

## How to install

Create a new Neutralinojs project with this template using the following command:

```bash
neu create myapp --template salehahmed99/neutralinojs-vite-tailwindcss
cd myapp

```

## How to develop

Start the Vite development server and the Neutralinojs app with HMR (Hot Module Replacement) enabled:

```bash
neu run

```

## How to bundle the app

Trigger a new production build and create the application bundle with the following command:

```bash
neu build

```
